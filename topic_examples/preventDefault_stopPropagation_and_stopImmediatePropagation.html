Event.preventDefault()

Prevents the default action of the event from happening


    document.querySelector('a').addEventListener('click', function(event) {
      event.preventDefault();
      console.log('Link clicked, but no navigation occurred');
    });

Event.stopPropagation()

prevents further propagation of the current event in the capturing and bubbling phases.


    document.getElementById('myElement').addEventListener('click', function(event) {
      event.stopPropagation();
      console.log('myElement clicked');
    });
    
    document.body.addEventListener('click', function() {
      console.log('body clicked');
    });

    // OUTPUT:
    // myElement clicked

The click event doesn't propagate to the body element because of event.stopPropagation().


Event.stopImmediatePropagation()

also prevents further propagation of the current event, but in addition, it also prevents any additional event handlers from being called on the same element.


    document.getElementById('myElement').addEventListener('click', function(event) {
      event.stopImmediatePropagation();
      console.log('first listener');
    });
    
    document.getElementById('myElement').addEventListener('click', function() {
      console.log('second listener');
    });

    // OUTPUT:
    // first listener
